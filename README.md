# uni-preset-vue-vite-ts

## 项目拉取

```bash
# 克隆项目
git clone https://github.com/Ace627/uni-preset-vue-vite-ts.git

# 进入项目目录
cd uni-preset-vue-vite-ts

# 安装依赖 | 建议不要直接使用 cnpm 安装依赖，会有各种诡异的 bug
pnpm i

# 启动服务
pnpm dev:mp-weixin

# 将 dist/dev/mp-weixin 文件夹导入微信开发者工具即可

```
